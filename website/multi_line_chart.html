Multi-line Chart

This visualization presents tobacco usage data over time. To make it even more insightful, we've added a sex selection feature, allowing you to choose between male, female, or both data. Each line represents a separate country, enabling you to track changes in tobacco use over time according to the selected sex. You can hover over each line to see more detailed data for each country. By understanding the trends and variations across different countries and sexes, we can gain valuable insights into the patterns and dynamics of tobacco consumption on a global scale.
